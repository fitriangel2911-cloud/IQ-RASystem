# IQ-RASystem
IQ-RA System adalah platform web koperasi syariah berbasis full-stack framework modern. Menghadirkan asisten AI (RAG) untuk rekomendasi akad serta otomatisasi laporan keuangan sesuai SAK EP dan PSAK Syariah 401-407. Solusi digital yang adaptif, transparan, dan akurat untuk ekosistem syariah.
